# macOS Packaging

Files in this directory are generated by `packaging/build.py`:

- `com.aems.agent.plist` — LaunchAgent for auto-start
- The `.app` bundle and `.dmg` are created in `dist/`

## Manual Installation

1. Open `AEMS-Agent.dmg`
2. Drag "AEMS Agent" to Applications
3. Copy `com.aems.agent.plist` to `~/Library/LaunchAgents/`
4. Run: `launchctl load ~/Library/LaunchAgents/com.aems.agent.plist`

## Notarization

For distribution outside the App Store:
```bash
codesign --deep --force --verify --verbose \
    --sign "Developer ID Application: ..." \
    "dist/AEMS Agent.app"

xcrun notarytool submit dist/AEMS-Agent.dmg \
    --apple-id "..." --team-id "..." --password "..."
```
