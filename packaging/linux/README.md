# Linux Packaging

Files in this directory are generated by `packaging/build.py`:

- `aems-agent.desktop` — Desktop entry for auto-start
- `aems-agent.service` — systemd user service

## AppImage

After running PyInstaller, create an AppImage:
```bash
# Install appimagetool
wget https://github.com/AppImage/AppImageKit/releases/download/continuous/appimagetool-x86_64.AppImage
chmod +x appimagetool-x86_64.AppImage

# Create AppDir structure
mkdir -p AppDir/usr/bin AppDir/usr/share/applications AppDir/usr/share/icons
cp -r dist/aems-agent/* AppDir/usr/bin/
cp packaging/linux/aems-agent.desktop AppDir/
ln -s usr/bin/aems-agent AppDir/AppRun

# Build
./appimagetool-x86_64.AppImage AppDir dist/AEMS-Agent.AppImage
```

## Debian Package

```bash
mkdir -p deb-pkg/DEBIAN
mkdir -p deb-pkg/usr/local/bin
mkdir -p deb-pkg/usr/share/applications
mkdir -p deb-pkg/usr/lib/systemd/user

cp -r dist/aems-agent/* deb-pkg/usr/local/bin/
cp packaging/linux/aems-agent.desktop deb-pkg/usr/share/applications/
cp packaging/linux/aems-agent.service deb-pkg/usr/lib/systemd/user/

cat > deb-pkg/DEBIAN/control << 'EOF'
Package: aems-agent
Version: 0.2.0
Section: utils
Priority: optional
Architecture: amd64
Maintainer: AEMS Project
Description: AEMS Local Bridge Agent
 Local filesystem access for AEMS exam PDFs.
EOF

dpkg-deb --build deb-pkg dist/aems-agent_0.2.0_amd64.deb
```
